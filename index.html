Handy Resources

W3Schools

Beginner-friendly tutorials for HTML, CSS, Bootstrap, and more. Your go-to reference.

w3schools.com

MDN Web Docs

The definitive reference for web tech. More detailed than W3Schools, great when you need the full picture.

developer.mozilla.org

VS Code

The code editor we use in class. Free, fast, and full of helpful extensions.

code.visualstudio.com

Bootstrap 5

The CSS framework we use for responsive websites. Grid, components, utilities.

getbootstrap.com

Coolors

Generate colour palettes in seconds. Great for picking your site's colour scheme.

coolors.co

Google Fonts

Free web fonts. Pick a font pair, copy the link tag, done.

fonts.google.com

Font Awesome

Icon library. Thousands of free icons you can use in any website.

fontawesome.com

Monkeytype

Typing speed test. Track your WPM and accuracy over time.

monkeytype.com

Excalidraw

Quick wireframes in the browser. Hand-drawn style, easy to share.

excalidraw.com

Figma

Industry-standard design tool. Free tier covers everything you need.

figma.com

Unsplash

Free high-quality stock photos. No attribution required.

unsplash.com

Pexels

Another great source of free stock photos and videos.

pexels.com
