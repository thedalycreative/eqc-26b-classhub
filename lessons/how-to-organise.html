The How · For Dwain & Ayu

Build it once.
Use it forever.

One simple shelf for everything you study — Web Dev and Cyber — set up so you always know exactly where things go.

The trick: give every subject the same folders.

You're studying two things — back end web dev and cyber security. So we split your shelf in two.

Then we give each subject the exact same set of folders. Same names, same order. That way you never have to stop and think "hmm, where does this go?" — your brain already knows.

Your folder shelf 👇

Make these on your Desktop. That's the whole thing.

Both subjects have Notes · Practice · Assessments · Cheat Sheets. Learn the pattern once and it works for any subject you ever study.

What goes where

Start Here
A simple note reminding you how the folders work. Read it when you forget.
Course Info
Your timetable, login details, important dates, who to email. The boring-but-vital stuff.
Notes
Anything you write down or save from a lesson. Name them by week.
Code & Practice
Web dev: the code you write while practising. One folder per exercise or mini-build.
Labs & Practice
Cyber: your hands-on labs, walkthroughs, and anything you try out yourself.
Assessments
Graded work you're building. Keep it here until it's done and handed in.
Cheat Sheets
Quick references you'll use again and again — commands, snippets, handy links.
Submitted
Drop a copy of anything you hand in here. Proof you did it, safe forever.

Name things so future-you isn't confused.

A folder is only tidy if you can read the name and know what's inside. Be boringly clear.
